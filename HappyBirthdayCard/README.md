# 🎉 Happy Birthday Card App

A simple and beautiful **Happy Birthday Card Android App** built using **Kotlin** and **Jetpack Compose**.

This app displays a birthday message over a background image with clean UI styling and smooth animations.

## 🚀 Features

- 🎂 Beautiful birthday greeting message
- 🖼 Background image support
- 🎨 Styled text with custom colors
- ✨ Smooth fade-in animation
- 📱 Built using modern Jetpack Compose UI

---

## 🛠 Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Android Studio**

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/pawan-25k/MindMatrix/HappyBirthdayCard.git
```

2. Open in **Android Studio**
3. Sync Gradle
4. Run on emulator or physical device
