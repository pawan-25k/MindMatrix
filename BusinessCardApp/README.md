# 💼 Business Card App

A modern and minimal **Business Card Android App** built using **Kotlin** and **Jetpack Compose**.

This app displays a personal business card layout with profile image, name, designation, and contact information using Material 3 components.

## 🚀 Features

- 👤 Circular profile image
- 🏷 Name and professional title
- 📞 Contact information section
- 🎨 Custom background color
- 🧩 Reusable composable components
- 📱 Built using Jetpack Compose

---

## 🛠 Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Android Studio**

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/pawan-25k/MindMatrix/BusinessCardApp.git
```

2. Open in **Android Studio**
3. Sync Gradle
4. Run on emulator or physical device
