# 🖼️Art Gallery Navigator – Android App

A simple Art Gallery Navigator built using **Kotlin** and **Jetpack Compose**.  
This app allows users to browse artworks using Next and Previous buttons with circular navigation logic.

---

## 🚀Features

- Display artwork image
- Show artwork title and artist name
- Navigate using Next and Previous buttons
- Circular (wrapping) navigation
- Built with Jetpack Compose
- Simple and clean UI layout

---

## ⚙️Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
